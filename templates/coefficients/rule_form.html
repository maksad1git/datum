Что такое правило?

Правило определяет способ агрегации данных для построения отчётов. Например: как группировать визиты, какие фильтры применять, как усреднять показатели.

Приоритет

Порядок применения правил (меньше число - выше приоритет). При конфликте правил применяется правило с наивысшим приоритетом.

Конфигурация

JSON-объект с параметрами:

  • method: Метод агрегации (SUM, AVG, MIN, MAX, COUNT)
  • group_by: Поля для группировки (регион, канал, бренд и т.д.)
  • filters: Условия фильтрации данных
Примеры правил:

1. Средняя цена по регионам: 
{
  "method": "AVG",
  "group_by": ["region"]
}
2. Сумма продаж по каналам: 
{
  "method": "SUM",
  "group_by": ["channel"],
  "filters": {
    "status": "completed"
  }
}
